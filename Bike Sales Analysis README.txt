# 🚴‍♂️ Bike Sales Analysis – Power BI Dashboard

## 📌 Overview
This project is a **Bike Sales Analysis Dashboard** built using **Power BI**. It provides interactive visualizations and insights into bike sales data, helping users explore sales performance across various dimensions such as **model**, **engine capacity**, **brand**, **price**, and **mileage**.

## 📊 Key Features
- **Sales Trends** by brand, model, and engine type  
- **Top-Selling Bikes** based on volume and revenue  
- **Price vs. Mileage Comparison** for buyer insights  
- **Interactive Filters** for customized data exploration  
- **Clean Layout & Visual Aesthetics** for intuitive analysis  

## 🛠️ Tools & Technologies
- **Power BI Desktop**
- Data Transformation using **Power Query**
- DAX for custom measures and KPIs

## 📁 File Included
- `Anshika -231302255- data visualisation assignment.pbix`  
  Contains the complete Power BI report with data visuals and analysis.

## 📈 Use Case
This dashboard can be useful for:
- **Dealerships** and **marketing teams** to assess product performance  
- **Buyers** looking to compare features across brands  
- **Analysts** examining trends in the two-wheeler market

## 🧠 Insights Gained
- Identification of best-performing bike models and brands  
- Relationship between pricing and mileage  
- Market preferences based on engine capacity
